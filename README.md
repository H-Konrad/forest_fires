# Forest Fires

This project investigates predictive modelling techniques for estimating the burned area of forest fires using a Forest Fires dataset. The analysis implements a wide range of regression approaches, including linear regression, polynomial regression, ridge regression, gradient descent, Huber regression, and proximal methods, to assess how different modelling choices handle the datasetâ€™s highly skewed target variable.

## Repository Structure

### functions
- `gd_huber.py` : Implementation of gradient descent huber loss
- `gd_linear.py` : Implementation of linear gradient descent
- `gd_ridge.py` : Implementation of ridge gradient descent
- `linear_regressions.py` : Implementation of linear regression
- `polyn_regression.py` : Implementation of polynomial regression
- `proximal_gd.py` : Implementation of proximal gradient descent
- `ridge_regressions.py` : Implementation of ridge regression
- `sorting.py` : Functions for sorting the dataset
